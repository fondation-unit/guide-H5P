Guess the Answer

Cette activité permet au lecteur de deviner la réponse avant d’appuyer sur le bouton de révélation de celle-ci. Si cette activité fonctionne principalement avec une image ou une vidéo, il est possible de l’utiliser en 100% texte.

- Exemple -

Prêtez-vous à cet exercice avec notre exemple sur H5P.

- Caractéristiques -

A savoir :

  • Média autorisé : Image ou vidéo uniquement
  • Les images doivent être de bonne qualité et dans la largeur et non la longueur.

Données à prévoir :

  • Un titre pour chaque image.
  • Une description pour chaque image.

Peut être utilisé...

  • Pour proposer une demonstration étape par étape sans passer par la vidéo
