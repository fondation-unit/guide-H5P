Image Sequencing

Dans cette activité il est proposé à l’utilisateur de remettre les images dans le bon ordre via un drag and drop. L’utilisateur peut à tout moment vérifier le placement des images et si elles ne sont pas dans le bon ordre il peut reprendre l’activité. Durant toute son activité il est aussi informé du temps passé dessus et du nombre de déplacements effectués. Cette activité peux donc proposer un entraînement de rapidité.

Peut être utilisé...

Cette activité propose aussi d'ajouter du son à chaque image. Il est donc possible d'utiliser cette activité pour organiser dans le bon ordre des audio. Néanmoins, il est obligatoire d'ajouter une image pour que l'activité fonctionne. Par défaut les informations sur le temps, le nombre total de mouvement et les boutons d'action sont en anglais, mais il est possible de les traduires dans la partie option et textes.

  • Média autorisé : Image et audio
  • Formule de math : En légende de l'image
  • Réglage : Il est possible de traduire tout les textes de l'exercice dans la partie "option et textes"
  • Autre : Cette activité n'est pas disponible dans des conteneurs (column, course presentation, interactive book)
