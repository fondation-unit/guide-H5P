- Find the Hotspot -

À ne pas confondre avec Find Multiple Hotspots ou Image Hotspots. Dans ce quiz, l’utilisateur doit trouver la zone de bonne réponse. Il est possible d’ajouter plusieurs zones de bonne réponse, mais le quiz se stoppe quand l’une d’elles est trouvée. Il n’existe pas d’indice visuel pour les zones cliquables. Donc l’utilisateur peut cliquer "à l'aveugle" mais il en sera averti par un feedbakck.

Exemple

Prêtez-vous à cet exercice avec notre exemple sur H5P.

Caractéristiques

A savoir :

  • Média autorisé : Uniquement l'image support
  • Feedback : Pour chaque réponse/zone vide
  • Dynamisme : L’apprenant clique sur une zone

Données à prévoir :

  • Une question
  • Une image support, de bonne qualité
  • Feedback pour chaque réponse et pour la zone vide
