- Image Slider -

Cet outil vous permet de créer une présentation de vos images sous forme de carrousel. Il est possible pour le lecteur d’afficher les slides en plein écran. Attention, il n’existe pas de légendes possibles sous les images. Seul un texte au survol est disponible.

Prêtez-vous à cet exercice avec notre exemple.

Caractéristiques

A savoir :

  • Média autorisé : Image
  • Information : La légende de l'image est disponible au survol

Données à prévoir :

  • Une description de chaque image pour l'accessibilité
  • Une legende

Peut être utilisé...

  • Pour mettre en page une série d'images
  • Présenter les outils d'une expérience
