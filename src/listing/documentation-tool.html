Documentation tool

Cette puissante activité permet d’accompagner l’utilisateur dans la rédaction d’un travail en le guidant étape par étape. Celui-ci peut se voir proposer à chaque étapes une nouvelle consigne et des champs de saisis pour une rédaction progressive. Il est aussi possible de proposer un travail de réflexion sur des objectifs à atteindre. A la fin de l’exercice l’utilisateur peut télécharger son travail.
NB : Cette activité peut-être aussi utilisé comme assistant de formulaire.

Peut être utilisé...

Utilisation basique de documentation tools, consiste à proposer un travail de rédaction, comme l’exemple ci-dessus, en plusieurs parties. Mais en proposant une seule partie de rédaction, l’activité documentation tools, devient plus proche des activités d’H5P de type dictation ou Essay. Ses activités n’étant pas disponible dans un conteneur (column, courses présentation et Interactive book ), documentation compense leurs absences. Sans oublier l’atout de cette activité : elle propose de garder, à l’utilisateur, localement son travail. Ce qui n’est rarement possible avec des activités sur H5P.

  • Média autorisé : Image
  • Formule de math : Oui
  • Bonus : Téléchargement du travail
  • Il n'est pas possible de rentrer les objectifs en amont de l'exercice. C'est à l'utilisateur seul de les définir lui-même.
