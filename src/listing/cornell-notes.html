Cornell Notes

Dans cette activité il est proposé à l'utilisateur de prendre des notes selon la Méthode Cornell créée par Walter Pauk. Cette méthode consiste à proposer deux colonnes : L'une pour la prise de notes, l'autre pour les mots-clés.
Dans l'activité H5P, il est proposé en plus à l'utilisateur un espace de rédaction pour faire un résumé.

Peut être utilisé...

Il est aussi possible de proposer cette activité avec un texte support ou de l’audio. Elle peut donc servir à une compréhension orale ou une compréhension de texte. Néanmoins il n’est pas possible de réduire ou d’augmenter le nombre d’espace de rédaction c’est-à-dire qu’il est reste toujours de trois espaces.

  • Média autorisé : Vidéo, texte et audio
  • Formule de math : Oui
  • Réglage : Il est possible de personnaliser les trois espaces de rédaction.
  • Autre : Cette activité n'est pas disponible dans des conteneurs (column, course presentation, interactive book)
